Zip64File - Users' Manual - Introduction

Introduction

Zip64File was developed by Enter AG for the Swiss Federal Archives as part of the SIARD (Software-Independent Archival of Relational Databases) project.

It serves the purpose of reading and writing files in ZIP64 format (e.g. SIARD files adhering to the standardized SIARD Format 2.1).

This document is the users' manual of Zip64File.
